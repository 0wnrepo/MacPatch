Content Sync

Content sync is only needed in a multi-server environment. If you are only running a master server, then no content sync configuration is needed. If you have a master server and one or more distribution servers then you'll want to enable the "Master Server Sync" on the master server and populate either the hostnames or IP Addresses in the "Hosts Allowed" field using comma separation. Make sure to increment the "Max Connections" every time a distribution server is added.


Master Server Config

Distribution Server

On the distribution servers you will want to configure the "Sync From Master Server" service. The default sync interval is 1800 seconds (30 Minutes). 

Advanced

Manually run a sync from a distribution server (This may take while, depending on the amount of content)

% sudo -u _appserver /Library/MacPatch/Server/conf/scripts/MPSyncContent.py --server [my.master.mp.server]
