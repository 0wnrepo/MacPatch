Tomcat Server

The Tomcat Server does not have very many options. This section allows you to stop and start the Tomcat server and set it to start on system boot.

This tab is where you decided which tomcat applications you would like to run. To enable the Admin console click the check box for "Enable Admin Console" and the same for "Enable Web Services".

The "Admin Console" image is a button to open the Admin Console in your browser.


Advanced

The Tomcat server serves up content on 3 different ports; 8080, 8443, 2600. SSL is enabled on 8443 and 2600. Port 8080 if redirected to 8443.

Any and all changes like ports etc. are set in the /Library/MacPatch/Server/apache-tomcat/conf/server.xml file.
