Web Server

The Web Server tab does not have very many options. This tab allows you to stop and start the MacPatch web server and set it to start on system boot.

The web server serves up content on 3 different ports; 80, 443, 2600. SSL is enabled on 443 and 2600.

Advanced

Any and all changes like ports etc. are set in the /Library/MacPatch/Server/Apache2/conf/extra/httpd-vhosts.conf file.
